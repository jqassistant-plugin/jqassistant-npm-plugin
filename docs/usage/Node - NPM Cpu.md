# `:NPM:Cpu` Node  
  
-> represents a cpu architecture and its type (supported / blocked)
  
## Properties  
  
| Name   | Description                                                                           |
| ------ | ------------------------------------------------------------------------------------- |
| `name` | name of the cpu architecture e.g. "arm"                                               |
| `type` | type of the cpu architecture - either supported (e.g. "arm") or blocked (e.g. "!arm") |
