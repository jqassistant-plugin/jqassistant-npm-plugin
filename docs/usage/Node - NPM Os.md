# `:NPM:Os` Node  
  
-> represents an operating system and its type (supported / blocked)
  
## Properties  
  
| Name   | Description                                                                 |
| ------ | --------------------------------------------------------------------------- |
| `name` | name of the operating system                                                |
| `type` | type of the os - either supported (e.g. "win32") or blocked (e.g. "!win32") |
